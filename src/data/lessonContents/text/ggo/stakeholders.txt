# Stakeholder Analysis Template (Plantilla de análisis de stakeholders)

## Purpose / Propósito
- Document roles, power, and interest of stakeholders using the “Stakeholder Analysis 2024” template. / Documentar roles, poder e interés de los stakeholders usando la plantilla “Análisis de stakeholders 2024”.
- Keep messages and actions bilingual to ease coordination with international teams. / Mantener mensajes y acciones bilingües para facilitar la coordinación con equipos internacionales.

## Objectives / Objetivos
- Classify each actor on the Power–Interest matrix with a justified position. / Clasificar a cada actor en la matriz Poder–Interés justificando su posición.
- Define key messages and preferred channels tailored to each quadrant. / Definir mensajes clave y canales preferidos adecuados a cada cuadrante.
- Plan engagement actions with clear deadlines and an associated risk log. / Planificar acciones de involucramiento con plazos claros y un registro de riesgos asociado.

## Activities / Actividades
### Quadrant Setup (Configuración de cuadrantes)
- Duplicate the sheet keeping Spanish labels while adding notes in English. / Duplicar la hoja manteniendo las etiquetas en español mientras se añaden notas en inglés.
- Classify by **Power (poder)** and **Interest (interés)**; document the justification in English while repeating the Spanish term from the PDF. / Clasificar según **Power (poder)** e **Interest (interés)**; documentar la justificación en inglés repitiendo el término español del PDF.
- Record quadrant changes and their **triggering event / evento detonante** in bilingual form. / Registrar cambios de cuadrante y su **evento detonante** en formato bilingüe.

### Messaging Cues (Mensajes clave)
- For each quadrant, note the communication objective in both languages (e.g., “Keep satisfied (**mantener satisfecho**)…”). / Para cada cuadrante, anotar el objetivo comunicativo en dos idiomas (p.ej., “Keep satisfied (**mantener satisfecho**)…”).
- Indicate the **preferred channel** in English with the Spanish label in parentheses. / Indicar el **canal preferido** en inglés con la etiqueta española entre paréntesis.

### Engagement Actions (Acciones de involucramiento)
- Translate the recommended verbs while keeping the Spanish original: “Co-create (**cocrear**)…”. / Traducir los verbos recomendados manteniendo el original español: “Co-create (**cocrear**)…”.
- Add deadlines or cadences in ISO format so they integrate with the planner view. / Añadir fechas límite o cadencias en formato ISO para integrarse con la vista del planificador.

## Assessment / Evaluación
- Submit the exported PDF matrix with bilingual notes. / Entregar la matriz exportada a PDF con notas bilingües.
- Include a risk log listing stakeholders with **low support / apoyo bajo** and the mitigation in English. / Incluir un registro de riesgos que liste stakeholders con **apoyo bajo** y la mitigación en inglés.
- Attach a meeting agenda showing each attendee in Spanish with their role in English. / Adjuntar una agenda de reunión donde cada asistente aparece en español con su rol en inglés.
