### Course Components (Componentes del curso)
- Course goals / Objetivos del curso:
  - Understand core distributed systems concepts. / Comprender conceptos fundamentales de sistemas distribuidos.
  - Identify patterns and design approaches for distributed services. / Identificar patrones y enfoques de diseño de servicios distribuidos.
  - Recognize key challenges: security, coordination, resource optimization. / Reconocer retos clave: seguridad, coordinación, optimización de recursos.
  - Develop practical skills through guided labs. / Desarrollar habilidades prácticas mediante laboratorios guiados.

### Roadmap and Requirements (Hoja de ruta y requisitos)
- Content roadmap / Programa de contenidos:
  1. Distributed systems fundamentals. / Fundamentos de sistemas distribuidos.
  2. Architectures, coordination, and contracts (SLA, QoS, Cloud). / Arquitecturas, coordinación y contratos (SLA, QoS, Cloud).
  3. Security and governance. / Seguridad y gobierno.
  4. Deployment and tooling (NATS, Prometheus, STEP, APISIX, Docker). / Despliegue y herramientas (NATS, Prometheus, STEP, APISIX, Docker).
- Labs / Laboratorios:
  - Design a service integrating queues, reverse proxy, authentication, and deployment. / Diseñar un servicio que integre colas, proxy inverso, autenticación y despliegue.
  - Work in teams of three people. / Trabajo en equipos de tres personas.
  - Recommended technologies: TypeScript or Go, Docker, Git, VS Code. / Tecnologías recomendadas: TypeScript o Go, Docker, Git, VS Code.
  - Submit the repository packaged as `.tgz`. / Entrega del repositorio empaquetado en `.tgz`.
- Requirements and recommendations / Requisitos y recomendaciones:
  - Prior knowledge of computer science, programming, and operating systems. / Conocimientos previos de informática, programación y sistemas operativos.
  - Suggested experience: TypeScript/Golang, Git, Linux, collaborative work. / Experiencia sugerida: TypeScript/Golang, Git, Linux, trabajo colaborativo.
  - Balance between in-person sessions and self-directed work. / Balance entre sesiones presenciales y trabajo autónomo.
